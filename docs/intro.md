# 一切的起点
:::quote[[法] 阿尔贝·加缪《西绪福斯神话》]

我让西绪福斯留在山下！人们总是看得见他的重负。西绪福斯教人以否定神祇举起巨石的至高无上的忠诚。他也断定一切皆善。这个从此没有主人的宇宙对他不再是没有结果和虚幻的了。这块石头的每一细粒，这座黑夜笼罩的大山的每一道矿物的光芒，都对他一个人形成了一个世界。征服登上顶峰的斗争本身足以充实人的心灵。**应该设想，西绪福斯是幸福的。**

:::
